# Stem: Story-driven job workspaces (project_tracker)

## Claim

Dashboard/board are job homes: metrics + open/unassigned queues before kanban density.

## Reconstruct

- Admin/manager default: dashboard = portfolio metrics + open task queue + grid + kanban.
- Member default: project_board = board metrics + kanban + unassigned queue.

## Not this

- Persona lands on a bare entity list when the job is triage, review, or oversight.
- Story `given:` workspace names that disagree with `default_workspace`.

## Expressions

- `dsl/` workspaces + personas; `docs/guides/story-to-composition.md`
