# Domain theory — support_tickets (falsifiable hypotheses)

Used by the agent QA ladder (`dazzle qa trial-hypotheses`, deep trials).
Mark each H-id confirmed | falsified | inconclusive after a run.

## Manager

- **H-M1** — Urgency of open/critical work is visible on Manager Ops without spreadsheet mental math.
- **H-M2** — Manager can locate unassigned work and start a reassignment path in under a minute from the primary workspace.
- **H-M3** — Empty or filtered queues explain *why* they are empty (not a blank wall).

## Customer (signing)

- **H-C1** — Customer can complete SLA waiver sign/decline from inbox without staff help.
- **H-C2** — Expired token surface is honest (not a silent happy path).
- **H-C3** — Already-signed re-open shows completion + download, not a second blank form.

## Agent

- **H-A1** — Agent sees only their actionable queue, not manager-only pressure chrome.
