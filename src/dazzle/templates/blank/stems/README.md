# Stems — this project

Compressed **domain** judgement for this Dazzle app. Framework judgement lives
in the Dazzle monorepo `stems/` (or package docs when using a release).

## How to use

1. Edit [INDEX.md](INDEX.md) as stems emerge (personas, non-goals, core entities).
2. One short `*.md` per stem: Claim / Reconstruct / Not this / Expressions.
3. Link SPEC.md and `dsl/` as **expressions**, not as competing stems.
4. Agents: read `stems/INDEX.md` after `AGENTS.md` when changing domain behaviour.

## Do not

- Copy framework stems into this folder (link them).
- Dump the entire SPEC here — compress.
