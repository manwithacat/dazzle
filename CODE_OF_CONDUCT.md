# Code of Conduct

## Summary

Be professional. Be respectful. Focus on the code.

## Expected Behavior

- Constructive technical feedback
- Respectful disagreement
- Helping others learn
- Accepting feedback gracefully

## Unacceptable Behavior

- Personal attacks or harassment
- Trolling or inflammatory comments
- Spam or off-topic content
- Sharing others' private information

## Enforcement

Violations may result in content removal or blocked participation at maintainer discretion.

## Scope

This applies to all project spaces: issues, PRs, discussions, and related communications.

## Contact

Report issues to the repository owner via GitHub.
